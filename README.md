# HR_Analysis_Dashboard
Built an interactive HR analytics dashboard to analyze employee attrition trends and identify key factors affecting workforce retention using Power BI.
📊 HR Attrition Analysis Dashboard (Power BI)
📌 Project Overview

This project focuses on analyzing employee attrition using HR analytics data and building an interactive Power BI dashboard to uncover key trends and drivers of workforce turnover.

The dashboard provides a comprehensive view of employee behavior, helping identify factors influencing attrition such as salary, overtime, tenure, and job role.

🎯 Objectives
Analyze employee attrition patterns across different dimensions
Identify key drivers impacting employee turnover
Provide actionable insights to improve retention strategies
Build an interactive dashboard for easy data exploration
🛠️ Tools & Technologies
Power BI – Data visualization & dashboard creation
Microsoft Excel / CSV – Data source
📈 Key Features
KPI tracking (Attrition Rate, Average Salary, Average Tenure)
Attrition analysis by:
Department
Job Role
Salary Band
Age Group
Gender & Marital Status
Identification of key attrition drivers:
Low compensation
Overtime workload
Early tenure employees
Interactive filters (slicers) for dynamic analysis
Drill-down capabilities for deeper insights
🔍 Key Insights
Employees in lower salary bands show higher attrition rates
Overtime is strongly associated with increased employee turnover
Majority of attrition occurs within the first few years of employment
Sales department and certain job roles experience higher attrition

💡 Business Recommendations
Review compensation structure for low salary bands
Monitor and manage employee overtime to reduce burnout
Improve onboarding and engagement for early-tenure employees
Focus retention strategies on high-attrition departments

Download the .pbix file
